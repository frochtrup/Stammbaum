Hilfe & Anleitung
Stammbaum PWA · Version 3.0

GEDCOM-Datei öffnen
Tippen Sie auf „GEDCOM-Datei öffnen" auf dem Startbildschirm — oder wählen Sie im Menü ☰ → „Öffnen…". Auf dem iPhone navigieren Sie im Dateipicker zu iCloud Drive → Genealogie → Ihre .ged-Datei.

Speichern — Chrome (Mac)
Beim Öffnen fragt Chrome nach Schreibzugriff auf die Datei — bestätigen. Danach speichert ☁️ direkt in die Originaldatei, ohne Download-Dialog.

Speichern — Safari / Firefox (Mac)
☁️ lädt die aktualisierte Datei herunter. Bei Änderungen wird zusätzlich ein Zeitstempel-Backup des Originals erzeugt. Tipp: Safari → Einstellungen → Allgemein → Downloadordner auf iCloud Drive/Genealogie setzen.

Speichern — iPhone / iPad
☁️ öffnet das Share Sheet mit der aktualisierten Datei + Zeitstempel-Backup. Wählen Sie „In Dateien sichern" → iCloud Drive → Genealogie.

Navigation
6 Tabs in der unteren Leiste: ⧖ Baum · 👤 Personen · ⚭ Familien · 📖 Quellen · 📍 Orte · 🔍 Suche. Die Suche durchsucht gleichzeitig alle Entitätstypen. Aus der Detailansicht gelangen Sie mit „← Zurück" wieder zur Liste.

Fotos & Medien
In jeder Detailansicht (Person, Familie, Quelle) finden Sie einen Abschnitt „Medien". Über „+ Hinzufügen" fügen Sie neue Einträge hinzu — bei aktiver OneDrive-Verbindung können Sie direkt eine Datei im Ordner auswählen. Mit dem ×-Button entfernen Sie Einträge. Tippen auf einen Medieneintrag öffnet das Foto in der Vollbild-Lightbox; dort können Sie es über „Als Hauptfoto setzen" zum Profilfoto machen.

OneDrive-Fotos
Im Menü ☰ → „🖼 Foto-Ordner einrichten" verbinden Sie einen OneDrive-Ordner mit Ihren Fotos. Die App verknüpft die Dateien automatisch mit den richtigen Personen (anhand der OBJE-Referenzen im GEDCOM) und lädt sie bei Bedarf dynamisch — ohne vollständigen Download.

Datenschutz
Alle Daten bleiben auf Ihrem Gerät. GEDCOM-Daten und Fotos werden nicht an Server übertragen. OneDrive-Zugriff erfolgt direkt zwischen Ihrem Browser und Microsoft — nur wenn Sie sich explizit verbinden.
