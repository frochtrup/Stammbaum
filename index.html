Hilfe & Anleitung
Stammbaum PWA · Version 2.0 · Phase 2

GEDCOM-Datei öffnen
Tippen Sie auf „GEDCOM-Datei öffnen" auf dem Startbildschirm — oder wählen Sie im Menü ☰ → „Öffnen…". Auf dem iPhone navigieren Sie im Dateipicker zu iCloud Drive → Genealogie → Ihre .ged-Datei.

Speichern — Chrome (Mac)
Beim Öffnen fragt Chrome nach Schreibzugriff auf die Datei — bestätigen. Danach speichert ☁️ direkt in die Originaldatei, ohne Download-Dialog.

Speichern — Safari / Firefox (Mac)
☁️ lädt die aktualisierte Datei herunter. Bei Änderungen wird zusätzlich ein Zeitstempel-Backup des Originals erzeugt. Tipp: Safari → Einstellungen → Allgemein → Downloadordner auf iCloud Drive/Genealogie setzen.

Speichern — iPhone / iPad
☁️ öffnet das Share Sheet mit der aktualisierten Datei + Zeitstempel-Backup. Wählen Sie „In Dateien sichern" → iCloud Drive → Genealogie.

Navigation
Nutzen Sie die untere Navigationsleiste um zwischen Baum, Personen, Familien, Quellen und Orten zu wechseln. Aus der Detailansicht gelangen Sie mit „← Zurück" wieder zur Liste.

Datenschutz
Alle Daten bleiben auf Ihrem Gerät. Es wird nichts hochgeladen oder geteilt.
